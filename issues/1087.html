/etc/localnetworks Local networks file

Description

To simplify /etc/hosts.{allow,deny} templates an external network list file can be included. Form hosts.allow manpage:

A string that begins with a ‘/´ character is treated as a file name. A host
name or address is matched if it matches any host name or address pattern listed in the named file. The file format is zero or more lines with zero or more host name or address patterns separated by whitespace. A file name pattern can be used anywhere a host name or address pattern can be used
