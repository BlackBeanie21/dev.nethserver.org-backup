Ntopng doesn't start

Added by Giacomo Sanchietti over 6 years ago. Updated over 6 years ago.

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

Security class: Resolution:
Affected version:v6.6-final NEEDINFO:No

Description

After an upgrade from 6.5 ntopng can't start if /usr/local/bin/ntopng link doesn't exists.

Steps to reproduce:
  • Execute:
    rm -f /usr/local/bin/ntopng
start ntopng
  • ntopng is not started

Expected behavior:
ntopng must start

The problem is present only with ntopng-1.2.2-8637.x86_64 rpm.
