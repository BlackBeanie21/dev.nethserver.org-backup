See Dovecot documentation:
http://wiki2.dovecot.org/Logging

To avoid debug messages in /var/log/imap we could tweak rsyslogd configuration under /etc/rsyslog.d/imap.conf, or try to not send any debug message at all, by setting debug_log_path = /dev/null in Dovecot configuration. The latter is only an hypothesis, I don't know if it can work.
