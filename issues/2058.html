ObjectsCollection UI widget

Resolution: NEEDINFO:No

Description

An ObjectsCollection UI widget displays a collection of objects, put into the view as an array of hashes.

The ObjectsCollection has the following attributes:

  • template
    specifies a view template that renders each member of the collection
  • id
    the hash key that identifies uniquely an object of the collection
  • ifEmpty
    the template instantiated when the collection is empty
