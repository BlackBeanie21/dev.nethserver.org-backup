Description

The POP3 connector page does not list all groups.
  • A group is not listed if it has "Send a copy of the message to group members" option .
  • Shared folder group are listed correctly.

Moreover, mail messages are delivered to the mailbox of the group, instead of being copied and delivered to each group member.
