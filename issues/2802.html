Roundcube: installation fails

Description

The installation can fail if the configuration script can't DROP the roundcube user.

Change nethserver-roundcubemail-conf action to use this syntax:

GRANT ALL ON myuser.* TO mydb@localhost  IDENTIFIED BY 'mypass';
FLUSH PRIVILEGES;
