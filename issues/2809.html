Firewall: web interface for policy routing

Added by Giacomo Sanchietti about 7 years ago. Updated over 5 years ago.

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

100%

Category:nethserver-firewall-base
Target version:v6.7
Resolution: NEEDINFO:No

Description

Create a new interface module for managing policy routing rules.

Using the UI, the user should be able to:
  • Create/edit a rule
  • Select source and destination using object picker
  • Select service using object picker
  • Re-order the rule list
For policy routing rules format see:
  • http://docs.nethserver.org/projects/nethserver-devel/en/latest/gateway.html#force-traffic-to-a-specific-provider
  • Issue #2740
