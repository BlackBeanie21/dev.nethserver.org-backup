mail-common/smarthost: SASL authentication failure

Added by Davide Principi over 6 years ago. Updated about 6 years ago.

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

100%

Category:nethserver-mail-common
Target version:v6.6
Security class: Resolution:
Affected version:v6.6-final NEEDINFO:No

Description

If the Firewall/SMTP Proxy module is installed (no mail server), sending mail through a smarthost fails. In /var/log/maillog:

May  5 12:20:11 nethsec default/smtp[17102]: warning: SASL authentication failure: No worthy mechs found

Workaround

yum install cyrus-sasl-plain
