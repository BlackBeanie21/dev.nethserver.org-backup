Description

Nethserver lacks of a PBX add-on.
Nethserver Enterprise has a PBX add-on. We should like to have the core part of this add-on also for community version

What we need:
  • Asterisk
  • FreePBX
  • a nethserver-voice configuration package
  • dahdi-linux package
  • sound and voices packages

We need an rpm package that only places FreePBX files somewhere and configuration stuff will be done by actions in configuration packages
