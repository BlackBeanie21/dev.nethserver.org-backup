Enhancement #2638

Translator fallback language

Added by Davide Principi over 7 years ago. Updated over 7 years ago.

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

100%

Category:nethserver-nethgui
Target version:v6.5-rc1
Resolution: NEEDINFO:No

Description

Define a fallback language chain for the translate() method. If a label is not found for the requested language, try to retrieve it from the fallback language chain. At least en language must be defined.

Implement also a general switch to disable the fallback mechanism: the current behaviour (showing the label itself if translation is not found) is convenient during development.
