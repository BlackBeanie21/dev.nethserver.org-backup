Package Manager: new UPDATE button and optional packages selection

Added by Davide Principi over 6 years ago. Updated over 6 years ago.

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

100%

Category:nethserver-base
Target version:v6.6-beta1
Resolution: NEEDINFO:No

Description

  • Add an UPDATE button to the Package Manager page. Clicking on it lists available software updates for confirmation. It's a yum update equivalent.
  • The optional packages of each module are now selectable from the Review page: move them in the main Select view.
