Table view: sort DataTable by file-size, percent, ip-address. Refs #2882

The sorting function can be set by passing an array for the column in
setColumns(). For instance

$columns = array(
    'Key',
    array(
            'name' => 'size',
            'columnDefs' => array('type' => 'file-size')
        ),
    ...
    );
