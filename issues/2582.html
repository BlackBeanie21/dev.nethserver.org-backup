Renamed event first-boot to system-init.

New action nethserver-base-initialize-db configures:
  • timezone
  • DNS
  • FQDN
  • green interface
The script parses all ifcfg-* scripts and stores information on network database. If no green interface can be found, the scripts parses ifconfig output to determinate current ip address. If no ip address can be found the default configuration will be:
  • ip: 192.168.1.100
  • netmask: 255.255.255.0
  • gateway: 192.168.1.1

The green interface is the one used to reach the default gateway.

The value for gateway prop is searched in 3 different places:
  1. First the script parses /etc/sysconfig/network
  2. If not found, the script will parse ip route command output (this is the case of network configured in dhcp)
  3. If not found, the script will use the one on /etc/sysconfig/network-scripts/ifcfg-ethX
  4. If not found, the default is 192.168.1.1
