Bug #2214

Group Shared Folder always shown

Added by Massimo Palazzetti almost 8 years ago. Updated over 7 years ago.

Status:CLOSEDStart date:11/14/2013
Priority:NormalDue date:11/14/2013
Assignee:-% Done:

100%

Category:nethserver-mail-server
Target version:v6.5-beta3
Security class: Resolution:
Affected version:v6.4-beta1 NEEDINFO:No

Description

The problem
Using Thunderbird 17 ESR the Shared folder was shown if the user is part of a group with the email service activated.

How To Replicate
  1. Create a group
  2. Select the "Email" checkbox and "Send a copy of the message to group members" radio button
  3. Add the interested user to the group

Check the Thunderbird folder list (of the selected user): as a result the folder "Shared" was created.
