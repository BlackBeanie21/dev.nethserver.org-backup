Bug #1783

Alpha2 ISO includes packages from CentOS 6.3

Added by Davide Principi over 8 years ago. Updated about 8 years ago.

Description

The NethServer 6.4 alpha2 ISO, released on March 21, 2013 carries RPMs from CentOS 6.3.

The following packages wants to be updated from CentOS 6.4 DVD1 RPM set:

Fix¶

To update the packages type as root:

   # yum --disablerepo=centos-updates update

The problem is probably originated by build-iso command in nethserver-devbox package.
