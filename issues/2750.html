Feature #2750

Mail spying / always Bcc

Added by Davide Principi about 7 years ago. Updated almost 7 years ago.

Status:CLOSEDStart date:
Priority:NormalDue date:
Assignee:-% Done:

100%

Category:nethserver-mail-common
Target version:v6.5
Resolution: NEEDINFO:No

Description

The "hidden copy" checkbox in Mail > Domains copies any message delivered to a local domain to the given mail address. This can be useful for backups of incoming messages, but does not catch outgoing.

Add an "always Bcc" to Mail > Messages to secretly copy any message received/sent by the mail system.
