Upgrading from NethServer 6.5 rc1¶

Update the system in the usual way:

   # yum clean all
   # yum update

Perform cleanup tasks:
  • Fetchmail group nethserver-fetchmail
  • Default spam thresholds in nethserver-mail-filter
