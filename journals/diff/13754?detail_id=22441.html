Bug #3006

When opening an inline help, the page displays "Internal server error".

Steps Step to reproduce:
* Install nethserver-backup-config
* Go to the web UI and open the Backup configuration page
* Click on the Help button
* The error is displayed on the right

Expected result:
* The inline help must be correctly displayed

The bug is present only in help pages with submodules, this is caused by a missing dependency.
