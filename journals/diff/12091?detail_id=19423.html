Feature #2748

The upgrade requires this effort:
* download and push new sogo rpms (2.2.7) (2.2.4)
* update nethserver-sogo rpm
** No special measure needs to be taken when upgrading from v2.1.1b v2.2.4
** enable supports for Microsoft ActiveSync protocol (see sogo installation guide) and test it with outlook 2013
** enable email reminders (see docs)
* update sogo-frontends rpm with new frontends (24.0.6) (24.0.4)
